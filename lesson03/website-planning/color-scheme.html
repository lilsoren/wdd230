The primary color is orange which implies a friendly, fun time is ahead.

Secondary color #0050a6

The secondary color is blue because it has a high contrast with orange and will add to the "pop-of-color" effect which draws the attention of the reader.

The typeography color is dark grey to lessen the contrast between the background. This will help not strain the users eyes when they are on the site.
